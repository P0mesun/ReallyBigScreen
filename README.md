# ReallyBigScreen

> 中文名：如意数据大屏

ReallyBigScreen（如意数据大屏）是一个公开的数据可视化大屏学习项目，目标是帮助大家从 0 到 1 学习如何自己动手制作一个数据可视化大屏。

## 项目定位

本项目面向希望学习数据大屏开发的同学、前端开发者、数据可视化爱好者以及需要搭建展示型数据看板的团队。项目会以持续迭代的方式，逐步补充从基础页面搭建、布局设计、图表接入、数据联动到最终部署上线的完整实践内容。

## 项目目标

- 从零开始构建一个数据可视化大屏项目
- 梳理数据大屏开发中的常见技术点和工程实践
- 提供可学习、可复用、可扩展的项目结构
- 帮助初学者理解数据大屏从设计到实现的完整过程

## 后续计划

后续将持续维护并逐步增加以下内容：

- 大屏基础布局
- 响应式适配方案
- 可视化图表组件
- 动态数据接入
- 大屏主题与动效
- 项目部署说明
- 从 0 到 1 的教程文档

## 适合人群

- 想学习数据可视化大屏开发的初学者
- 希望了解前端可视化项目工程化实践的开发者
- 需要参考数据大屏项目结构的学习者
- 对数据展示、图表设计、可视化交互感兴趣的同学
